## 💡 학습 팁

1. **완벽주의 버리기**: 100% 이해하고 넘어가려 하지 말기
2. **직접 타이핑**: 복사-붙여넣기 말고 직접 작성
3. **작은 단위로**: 큰 기능을 작은 단위로 나누기
4. **질문하기**: 30분 고민해도 안 되면 질문
5. **복습하기**: 주말에 한 주 내용 정리
6. **꾸준히**: 매일 조금씩이라도
