# 오늘 뭐하지? (What To Do)

날씨 + 한강 수온 확인 및 날씨 기반 활동 추천 React Native 앱

## 기술 스택

- Expo SDK 54
- expo-router v6
- TanStack Query v5
- ky
- React Native Animated API + Reanimated 4
- expo-location

## 실행 방법

```bash
npm install
npm start
npm run ios
npm run android
```

## 프로젝트 구조

```text
app/           # 스크린 (expo-router)
components/    # UI 컴포넌트
hooks/         # 커스텀 훅
services/      # API 호출 함수
lib/           # 설정
constants/     # 상수
types/         # 타입 정의
```
