## 오늘 배운 것

### React Native 실행하는 방법 2가지

- Expo
- React Native CLI

### Expo

- Expo Go 앱을 설치하면 별도의 개발 환경 설정 없이 바로 앱을 실행해볼 수 있음
- QR 코드를 스캔하여 앱을 실행함
- 플랫폼 지원 : 안드로이드, iOS, TV, 웹
- 장점 : 간편함, 빠른 개발 속도
- 단점 : 네이티브 기능 사용 제한, 앱 크기 증가

### React Native CLI

- 프레임워크 없이 네이티브 앱을 개발하는 방법
- Android Studio, Xcode 설치 필요
- 플랫폼 지원 : 안드로이드, iOS, TV, 웹, watchOS, macOS, visionOS, 윈도우
- 장점 : 네이티브 기능 사용 가능, 앱 크기 감소
- 단점 : OS별 개발 환경 설정 필요

### Expo로 React Native 앱 실행

```bash
# 프로젝트 생성
npx create-expo-app@latest

# 프로젝트 초기화
npm run reset-project

# 프로젝트 실행
npm run android // Android SDK 필요
npm run ios // Xcode 필요
npm run web // 웹 브라우저 실행 - port 8081
```
