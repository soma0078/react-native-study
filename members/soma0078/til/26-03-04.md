## 1. Layout & Tab 네비게이션

### `_layout.tsx`의 역할

- `Slot`: 자식 컴포넌트를 그대로 렌더링
- `Stack`: 자식을 Stack Navigator로 감싸서 렌더링 → 화면 쌓기 방식, 뒤로가기 자동 생성
- `Tabs`: 하단 탭 바를 통해 화면 전환

### Tabs vs NativeTabs

- `Tabs`: 크로스플랫폼 동일한 탭 UI (커스텀 스타일 자유도 높음)
- `NativeTabs`: 플랫폼별 네이티브 탭 UI (iOS는 UITabBar, Android는 네이티브 탭)
- 플랫폼별 분기:
  - `_layout.native.tsx` → iOS/Android에 NativeTabs 적용
  - `_layout.tsx` → 웹에 Tabs 적용

### Tabs 아이콘 설정

```tsx
<Tabs.Screen
  name="profile"
  options={{
    title: "Profile",
    tabBarIcon: ({ focused, color }) => (
      <FontAwesome
        name={focused ? "user-circle" : "user-circle-o"}
        size={20}
        color={color}
      />
    ),
  }}
/>
```

- `focused`: 현재 탭이 활성화 상태인지 여부 → 아이콘 변경에 활용
- `color`: 탭바의 `tabBarActiveTintColor` / `tabBarInactiveTintColor` 자동 적용

## 2. SectionList

섹션별로 데이터를 그룹화하고 헤더와 함께 렌더링할 때 사용

```tsx
<SectionList
  sections={SKILLS} // { title, data }[] 형태
  keyExtractor={(_, index) => String(index)}
  ListHeaderComponent={<ProfileHeader />} // 리스트 상단 고정 컴포넌트
  renderSectionHeader={({ section: { title } }) => <Text>{title}</Text>}
  renderItem={({ item, section }) => <Text>{item}</Text>}
  contentContainerStyle={styles.listContent} // 리스트 전체 컨테이너 스타일
/>
```

## 3. Expo Icons (@expo/vector-icons)

Expo에서 제공하는 아이콘 라이브러리

```tsx
import FontAwesome from "@expo/vector-icons/FontAwesome";
import Octicons from "@expo/vector-icons/Octicons";
import MaterialCommunityIcons from "@expo/vector-icons/MaterialCommunityIcons";

<FontAwesome name="user-circle" size={20} color="orange" />;
```

- 여러 아이콘 셋을 지원: `FontAwesome`, `Octicons`, `MaterialCommunityIcons`, `Ionicons` 등
- `focused` 상태에 따라 아이콘을 다르게 설정하는 패턴이 탭바에서 기본적으로 쓰임
- 아이콘 탐색: [icons.expo.fyi](https://icons.expo.fyi/Index)

## 4. ImageBackground & Image

로컬 이미지는 `require()`, 네트워크 이미지는 `{ uri: "..." }` 형태

### Image

```tsx
<Image
  source={require("../assets/images/duck.jpeg")}
  style={styles.profileImg}
/>
```

### ImageBackground

```tsx
<ImageBackground
  source={{ uri: "https://picsum.photos/seed/cloudy/800/400" }}
  style={styles.bgImg}
  resizeMode="cover"
/>
```

- `Image`와 달리 자식 컴포넌트를 이미지 위에 올릴 수 있음
- `resizeMode`: 이미지 크기 조절 방식
  - `cover`: 비율 유지하며 영역을 꽉 채움
  - `contain`: 비율 유지하며 영역 안에 맞춤
  - `stretch`: 비율 무시하고 영역에 맞게 늘림
