## React Native (Expo) 프로필 화면 구현

### 사용한 RN 컴포넌트 / 개념

| 컴포넌트·API         | 설명                                                                                                    |
| -------------------- | ------------------------------------------------------------------------------------------------------- |
| **View**             | 레이아웃용 컨테이너. div처럼 자식들을 감싸고, `style`로 배치·정렬.                                      |
| **Text**             | 텍스트 전용. RN에서는 문자열은 반드시 `Text` 안에 있어야 함.                                            |
| **Image**            | 이미지 표시. `source`에 `require(경로)` 또는 URI 전달.                                                  |
| **TouchableOpacity** | 터치 가능한 영역. 눌렀을 때 투명도 변화 + `onPress` 콜백 실행. 버튼·링크 역할.                          |
| **Linking**          | 외부 URL 열기. `Linking.openURL(url)` 로 브라우저·앱 실행.                                              |
| **StyleSheet**       | 스타일 객체 생성. `StyleSheet.create({ ... })` 로 스타일 정의 후 컴포넌트에 `style={styles.이름}` 전달. |

### 1. `_layout.tsx` (루트 레이아웃)

- **expo-router**의 `Stack`으로 화면 구조 정의
- `index` 화면에 "My Profile" 타이틀 설정

```tsx
import { Stack } from "expo-router";

export default function RootLayout() {
  return (
    <Stack>
      <Stack.Screen name="index" options={{ title: "My Profile" }} />
    </Stack>
  );
}
```

### 2. `index.tsx` (메인 프로필 화면)

- **구성 요소**
  - **프로필 이미지**
  - **이름**
  - **GitHub 배지**: `TouchableOpacity` + `FontAwesome` GitHub 아이콘, 탭 시 GitHub 링크 오픈 (`Linking.openURL`)
  - **직무**
  - **소개 말풍선**

```tsx
export default function Index() {
  const openGithub = () => {
    Linking.openURL("https://github.com/juseungyeon");
  };

  return (
    <View style={styles.container}>
      <View style={styles.imageCard}>
        <Image
          source={require("../assets/images/my-profile.jpeg")}
          style={styles.profileImage}
        />
      </View>

      <Text style={styles.nameText}>주승연</Text>
      <TouchableOpacity onPress={openGithub} style={styles.githubBadge}>
        <FontAwesome name="github" size={14} color="#fff" />
        <Text style={styles.githubText}>@juseungyeon</Text>
      </TouchableOpacity>

      <Text style={styles.jobText}>Frontend Developer / RN Beginner</Text>

      <View style={styles.bioBubble}>
        <Text style={styles.bioText}>
          {`안녕하세요! 👋 \n새로운 기술을 배우는 것을 즐기는 개발자입니다. ...`}
        </Text>
      </View>
    </View>
  );
}
```

### 3. `+not-found.tsx` (404 화면)

- 존재하지 않는 경로 접근 시 표시되는 화면
- 중앙 정렬된 "Not Found" 텍스트

```tsx
export default function NotFoundScreen() {
  return (
    <View style={styles.container}>
      <Text style={styles.text}>Not Found</Text>
    </View>
  );
}
```

### 사용한 API/라이브러리

- `react-native`: `Text`, `View`, `Image`, `StyleSheet`, `TouchableOpacity`, `Linking`
- `@expo/vector-icons`: `FontAwesome` (GitHub 아이콘)
- `expo-router`: `Stack`, 화면 라우팅
