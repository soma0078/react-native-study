## 오늘 배운 것

### 1. Expo 프로젝트 실행하기

- **생성**: `npx create-expo-app@latest`
- **실행** (프로젝트 폴더에서):
  - `npm run android` — Android 에뮬레이터
  - `npm run ios` — iOS 시뮬레이터
  - `npm run web` — 웹 브라우저
- **또는**: `npx expo start` 후 터미널에서 `i`(iOS), `a`(Android), `w`(웹), 또는 Expo Go 앱으로 QR 코드 스캔

---

### 2. 튜토리얼 프로젝트 전체 구조

```
my-app/
├── app/                    # 화면(페이지) 파일들
│   ├── _layout.tsx         # 앱 전체 레이아웃
│   ├── (tabs)/             # 탭 네비게이션 그룹 (URL에 안 나옴)
│   │   ├── _layout.tsx     # 탭 설정
│   │   ├── index.tsx       # Home 탭 화면
│   │   └── explore.tsx     # Explore 탭 화면
│   └── modal.tsx           # 모달 화면
├── components/             # 재사용 가능한 컴포넌트
├── constants/              # 테마, 색상 등 상수
├── hooks/                  # 커스텀 훅
├── assets/                 # 이미지, 폰트 등 리소스
└── scripts/                # 유틸리티 스크립트
```

---

### 3. 파일 기반 라우팅 (Expo Router)

`app/` 폴더 안의 **파일 이름이 곧 경로(URL)**. (Next.js와 비슷)

| 파일                     | 경로       |
| ------------------------ | ---------- |
| `app/(tabs)/index.tsx`   | `/` (홈)   |
| `app/(tabs)/explore.tsx` | `/explore` |
| `app/modal.tsx`          | `/modal`   |

- `(tabs)`처럼 **괄호로 감싼 폴더**는 URL에 안 나오는 "그룹". 탭으로 묶기 위한 용도.

---

### 4. `_layout.tsx` — 레이아웃

해당 폴더의 **모든 화면을 감싸는 공통 레이아웃**.

- **`app/_layout.tsx`** (루트)
  - `ThemeProvider` → 다크/라이트 모드
  - `Stack` → 화면이 카드처럼 쌓이는 네비게이션
  - `StatusBar` → 상단 상태바 스타일
- **`app/(tabs)/_layout.tsx`**
  - `Tabs` → 하단 탭바
  - `tabBarIcon` → 탭 아이콘
  - `HapticTab` → iOS 탭 누를 때 진동 피드백

---

### 5. React Native 스타일링 (웹과의 차이)

CSS 대신 **`StyleSheet.create()`**로 스타일 객체를 만듦.

```tsx
const styles = StyleSheet.create({
  titleContainer: {
    flexDirection: "row",
    alignItems: "center",
    gap: 8,
  },
});
```

| 웹              | React Native            |
| --------------- | ----------------------- |
| `<div>`         | `<View>`                |
| `<p>`, `<span>` | `<Text>`                |
| `<img>`         | `<Image>` (expo-image)  |
| `<a>`           | `<Link>` (expo-router)  |
| CSS 파일        | `StyleSheet.create()`   |
| React Router    | Expo Router (파일 기반) |
| `px`, `rem`     | 단위 없는 숫자 (dp)     |

- 기본 레이아웃은 **Flexbox** (`flexDirection` 기본값은 `column`)
- `className` 대신 `style` prop에 객체 전달
- **숫자만** 사용 (단위는 dp, 기기 독립 픽셀)

### 6. 주요 컴포넌트

| 컴포넌트                    | 역할                                           |
| --------------------------- | ---------------------------------------------- |
| `ThemedText` / `ThemedView` | 다크/라이트 모드에 따라 색상 자동 변경         |
| `ParallaxScrollView`        | 스크롤 시 헤더 패럴랙스 효과                   |
| `HelloWave`                 | 손 흔드는 애니메이션 (react-native-reanimated) |
| `HapticTab`                 | iOS 탭 촉각 피드백                             |
| `Collapsible`               | 접었다 펼 수 있는 아코디언                     |
| `ExternalLink`              | 앱 내 브라우저로 외부 링크                     |
| `IconSymbol`                | iOS: SF Symbols, Android/웹: Material Icons    |

### 7. 다크모드

- `useColorScheme()` 훅으로 현재 모드 감지 (`'light'` | `'dark'`)

### 8. 플랫폼별 코드

- **`Platform.select({ ios: ..., android: ..., default: ... })`** — 플랫폼별 값 선택
- **파일 확장자**: `icon-symbol.ios.tsx`는 iOS만, `icon-symbol.tsx`는 그 외 플랫폼에서 사용
